## Project Workflow

### Step 1: Database Design (SQL)
- Designed a normalized relational database to organize Union Budget data.
- Created master and mapping tables to reduce redundancy and maintain data integrity.
- Inserted and structured the dataset using SQL.

**Folder:** `SQL Data Insertion & Normalization`

---

### Step 2: Data Export & Preparation
- Exported normalized SQL tables.
- Cleaned and formatted the exported data in Excel.
- Merged relational tables into a single master dataset for analysis and reporting.

**Folder:** `Raw Exported Data`

---

### Step 3: Dashboard Development
- Built an interactive Excel dashboard from the master dataset.
- Implemented dynamic filtering using **INDEX-MATCH** and **Data Validation**.
- Added KPI cards and interactive feature extraction to help users explore budget benefits based on profession, age group, location, and other criteria.

**Folder:** `Excel Project`
